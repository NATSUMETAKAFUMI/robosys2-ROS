# ROSを使って日本標準時間を表示してみた

## 動作環境
---
### OS:ubuntu 20.04 server
### ROS:Noetic
---

## 実行方法
ROSのインストール後下記の手順を行ってください
'''sh
$ cd ~/catkin_ws/src
$ git clone https://github.com/NATSUMETAKAFUMI/robosys2-ROS.git
$ cd ..
$ catkin_make
$ source ~/.bashrc
'''
